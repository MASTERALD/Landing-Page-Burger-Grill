Burger Grill es una landing page desarrollada con HTML, CSS y JavaScript, pensada para mostrar de forma clara y directa la información de un restaurante. La página incluye una sección principal de presentación, un apartado para el menú, información básica sobre el negocio y un espacio para contacto. Todo el diseño está construido sin frameworks, buscando mantener un rendimiento ligero y una visual coherente tanto en computadoras como en dispositivos móviles.

El código está organizado para que sea fácil modificar estilos, textos o imágenes según se necesite. Para visualizar el proyecto no se requiere instalación de dependencias; basta con descargar los archivos y abrir el archivo index.html en un navegador. Cualquier sugerencia o mejora es bienvenida. El proyecto se comparte bajo la licencia MIT y puede utilizarse libremente para aprendizaje o referencia.
